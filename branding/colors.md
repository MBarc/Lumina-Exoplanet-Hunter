# Lumina — Brand Colors

## Primary Palette

| Name          | Hex       | Use |
|---------------|-----------|-----|
| Space Black   | `#04090f` | Page/app backgrounds |
| Deep Navy     | `#0d1f3c` | Card backgrounds, panels |
| Stellar Cyan  | `#00c8ff` | Primary accent — links, highlights, borders, charts |
| Star Gold     | `#ffd166` | Star elements, warnings, discovery highlights |
| Exo Green     | `#06d6a0` | Success states, confirmed candidates, planet elements |
| Nebula Purple | `#7b2fbe` | Secondary accent — badges, secondary actions |
| Light         | `#e8f4ff` | Body text on dark backgrounds |
| Muted         | `#4a6380` | Secondary text, disabled states |

## Usage Notes

- **Backgrounds**: Always Space Black or Deep Navy. Never white.
- **Primary CTA buttons**: Stellar Cyan fill, Space Black text
- **Danger/error**: `#ff4d6d`
- **Charts and light curves**: Stellar Cyan lines on Space Black
- **Planet/candidate highlights**: Exo Green
- **Typography**: System sans-serif stack — `'Trebuchet MS', 'Gill Sans', Arial, sans-serif`
- **Monospace (logs, data)**: `'Consolas', 'Courier New', monospace`

## Logo Colors

The logo uses the full palette:
- Background: Space Black + Deep Navy radial gradient
- Star: Star Gold → white core
- Planet atmosphere: Exo Green
- Network nodes + light curve: Stellar Cyan
- Network secondary nodes: Nebula Purple
- Text: Light (`#e8f4ff`) wordmark, Stellar Cyan tagline
