ABOUT THE MISSION

Lumina is a distributed citizen-science platform hunting for exoplanet transits in Kepler, K2, and TESS photometry. Contributors install the Lumina client on their machines, which automatically downloads light curves, runs the ExoNet neural network, and reports candidates back to the network.
